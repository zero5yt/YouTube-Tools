# 🚀 Almaras YT Mp3 & Video

🔥 Fast, Simple, and Powerful YouTube Downloader for Termux

Download your favorite YouTube videos as **MP3** or **MP4 (360p up to 1080p+)** directly using Termux.

---

## ✨ Features

* 🎵 Convert YouTube to **MP3 (High Quality)**
* 🎬 Download Videos:

  * 360p
  * 480p
  * 720p
  * 1080p
  * Best Quality
* ⚡ Fast download using yt-dlp
* 📂 Auto save to `Download/ATOOLS`
* 🖥️ Clean Terminal UI
* 📱 Termux Friendly
