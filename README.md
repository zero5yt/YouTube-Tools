# Almaras YT Mp3 & Video

Simple Termux-based YouTube downloader using yt-dlp.

## Features:
- MP3 High Quality
- Video 360p / 480p / 720p / 1080p
- Best Quality Option
- Auto save to Download/ATOOLS

## Install:
pkg update && pkg upgrade
pkg install python ffmpeg
pip install yt-dlp

## Run:
bash yt.sh
